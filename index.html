5 步驟申請 API 金鑰
只需幾分鐘即可完成設定，開始使用免費額度
1

前往 Google Cloud Console

開啟瀏覽器前往 console.cloud.google.com，使用 Google 帳號登入。

2

建立新專案

點擊頂部導覽列的專案選擇器 → 「新增專案」→ 輸入名稱後建立。

💡 建議為 TTS 單獨建立一個專案，方便管理費用和權限。
3

啟用 Text-to-Speech API

左側選單 → APIs & Services → Library → 搜尋「Cloud Text-to-Speech API」→ 點擊「啟用」。

4

建立 API 金鑰

APIs & Services → Credentials → Create Credentials → API Key

⚠️ 建議點擊「限制金鑰」，設定只允許呼叫 Text-to-Speech API，提升安全性。
5

啟用帳單（不會立即收費）

需要綁定信用卡才能啟用帳單，但在免費額度內不會收費。超過才開始計費。

import urllib.request, json, base64

API_KEY = "你的 API 金鑰"
TEXT = "你好，這是 Google Cloud Text-to-Speech 測試"

body = json.dumps({
    "input": {"text": TEXT},
    "voice": {
        "languageCode": "zh-TW",
        "name": "cmn-TW-Wavenet-A"  # 或 cmn-CN-Chirp3-HD-Zephyr
    },
    "audioConfig": {
        "audioEncoding": "MP3",
        "speakingRate": 1.0,   # 語速 0.25~4.0
        "pitch": 0.0           # 音調 -20~20
    }
}).encode()

req = urllib.request.Request(
    f"https://texttospeech.googleapis.com/v1/text:synthesize?key={API_KEY}",
    data=body,
    headers={"Content-Type": "application/json"}
)
response = urllib.request.urlopen(req)
data = json.loads(response.read())

# 儲存為 MP3
with open("output.mp3", "wb") as f:
    f.write(base64.b64decode(data["audioContent"]))

print("✅ 已儲存為 output.mp3")

const https = require('https');
const fs = require('fs');

const API_KEY = '你的 API 金鑰';
const body = JSON.stringify({
  input: { text: '你好，這是 Google Cloud TTS 測試' },
  voice: { languageCode: 'zh-TW', name: 'cmn-TW-Wavenet-A' },
  audioConfig: { audioEncoding: 'MP3' }
});

const options = {
  hostname: 'texttospeech.googleapis.com',
  path: `/v1/text:synthesize?key=${API_KEY}`,
  method: 'POST',
  headers: { 'Content-Type': 'application/json' }
};

const req = https.request(options, (res) => {
  let data = '';
  res.on('data', chunk => data += chunk);
  res.on('end', () => {
    const { audioContent } = JSON.parse(data);
    fs.writeFileSync('output.mp3', Buffer.from(audioContent, 'base64'));
    console.log('✅ 已儲存為 output.mp3');
  });
});
req.write(body);
req.end();
